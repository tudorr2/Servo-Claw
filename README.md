## ‚ö° Overview

ThermoClaw is a **handheld, 3D printed** servo claw controller built on Arduino. The enclosure and claw mechanism are fully 3D printed, housing dual servos driven from a single joystick axis with complementary motion logic ‚Äî while simultaneously monitoring ambient temperature via a **10KŒ© NTC thermistor** using the Steinhart-Hart B-parameter equation.

A future revision will expose all sensor data (servo positions, live temperature) through a **browser-based web interface** for remote monitoring and control.

> üîú **Next revision:** Claw fingers will be reprinted in **TPU 95A** (flexible filament) for a compliant, grippy surface that conforms around objects for a much more secure hold.

### Print Tips

- Print the body in **PETG** for better heat resistance near the electronics
- TPU 95A fingers: print slow (~25mm/s), disable retraction, 3‚Äì4 perimeters for durability
- The joystick and button should be accessible from the top face for one-handed operation

## üéÆ Control Logic

### Normal Mode

The joystick X-axis (`A5`) maps `0‚Äì1023` ‚Üí `0¬∞‚Äì180¬∞` for the drive servo.  
The steer servo runs **complementary** (mirrored): `abs(val - 180)`

```
Joystick 0%   ‚Üí  Drive: 0¬∞    Steer: 180¬∞
Joystick 50%  ‚Üí  Drive: 90¬∞   Steer: 90¬∞
Joystick 100% ‚Üí  Drive: 180¬∞  Steer: 0¬∞
```

### Button Mode (`D3` held LOW)

Both servos lock to **90¬∞** (neutral / center position).

## üßÆ Temperature Formula

Uses the **Steinhart-Hart simplified B-parameter equation**:


Where:

- `RT` = measured thermistor resistance
- `RT‚ÇÄ` = 10,000 Œ© (resistance at 25¬∞C)
- `B` = 3977 K (thermistor B-constant)
- `T‚ÇÄ` = 298.15 K (25¬∞C reference)

**Voltage divider resistance calculation (thermistor on top):**

```cpp

VR  = VCC - VRT;                           // voltage across fixed R
RT  = (VR / VRT) * R;                     // thermistor resistance
```

## üêõ Troubleshooting

| Symptom                       | Cause                                 | Fix                                          |
| ----------------------------- | ------------------------------------- | -------------------------------------------- |
| Temperature shows `-273.15¬∞C` | ADC reads 0 ‚Äî thermistor disconnected | Check wiring at `A2`                         |
| ADC reads `0` or `1023`       | Open circuit in voltage divider       | Verify both thermistor and 10KŒ© resistor     |
| Servo not moving              | Wrong pin or not attached             | Check `servodr.attached()` in Serial Monitor |
| Temperature jittery           | ADC noise                             | Add 100nF cap between `A2` and `GND`         |
